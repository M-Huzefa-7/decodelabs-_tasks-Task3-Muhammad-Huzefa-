# AI Recommendation System

## Overview
This project is a simple recommendation system that suggests items based on user preferences. It uses logic based matching to connect user interests with predefined categories.

## Features
- Takes user interest as input
- Matches input with predefined categories
- Displays recommended items
- Uses simple logic based recommendation

## How It Works
User enters an interest (e.g., action, comedy). The system matches it with a predefined dictionary and returns relevant recommendations.

## Technologies Used
- Python

## Learning Outcome
- Understanding recommendation systems
- Pattern matching logic
- User input handling
